# HooBank landing

Mobile-friendly landing page built with Vite + Tailwind CSS + React + TypeScript

## Installation

```bash
gh repo clone Ellinsa/bank-app
cd bank-app
```

```bash
npm install
```

```bash
npm run dev
```
